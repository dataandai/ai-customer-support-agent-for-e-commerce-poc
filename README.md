# AI Customer Support Agent POC

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Set up environment: `cp .env.example .env` and add your `OPENAI_API_KEY`.
3. Run the Backend: `python main.py`
4. Run the Frontend: `streamlit run app.py`

## Features
- **Order Tracking**: Ask 'Where is my order ORD-101?'
- **Policy Lookup**: Ask 'What is your refund policy?'
- **Product Search**: Ask 'Do you have any mice?'
